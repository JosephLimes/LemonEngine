void EditorScene::Initialize ( )

Initializes the scene.

This is called when the scene is set to be active by the SceneManager. Use this to set up any necessary objects, similar to a constructor.

Example

The following example shows how a Scene called Demo could use Initialize() to instantiate a GameObject called Character.

{
// set up GameObject
pMyGameObject = new Character();
