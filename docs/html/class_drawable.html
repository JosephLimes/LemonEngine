◆ Draw2D()

virtual void Drawable::Draw2D ( )
inlineprivatevirtual

Used to draw two-dimensional elements.

Called every frame once the object is registered. Draw 2D objects such as Sprite and SpriteString here.

Example

The following example shows how a GameObject called Character could use Draw2D.

void Character::Draw2D()
{
// draw a Sprite
pMySprite->Render()
// draw a SpriteString
pMySpriteString->Render();
}

Reimplemented in EditorAdmin, and EditorUI.

◆ SubmitRegistration()

void Drawable::SubmitRegistration ( )

Submit a request to the scene to recieve draw calls.

Once submitted, draw calls will begin at the top of the following frame.

Example:

The following example shows how a GameObject called Character could register for draw calls.

Character::Character()
{
// object construction
// register for draw calls
Drawable::SubmitRegistration();
}
